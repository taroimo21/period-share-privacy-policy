プライバシーポリシー / Privacy Policy

最終更新日 / Last Updated: 2026年1月17日 / January 17, 2026

収集する情報

本アプリは個人情報を収集しません。

保存されるテキストデータは全てお使いの端末内にのみ保存され、外部サーバーには送信されません。

キーボード拡張について

カスタムキーボード機能はフルアクセスを必要としません。

入力内容やクリップボードの内容が外部に送信されることはありません。

App Group機能を使用してメインアプリとキーボード間でデータを共有しますが、これは端末内での共有のみです。

Face ID / Touch ID

生体認証データはAppleのSecure Enclave内で処理され、本アプリがアクセスすることはありません。

認証結果（成功/失敗）のみを受け取ります。

データの保存場所

全てのデータは端末内のローカルストレージに保存されます。

iCloudへの同期やクラウドバックアップは行われません。

Information We Collect

This app does not collect any personal information.

All text data you save is stored only on your device and is never transmitted to external servers.

Keyboard Extension

The custom keyboard feature does not require Full Access.

Your input and clipboard contents are never sent externally.

App Group is used to share data between the main app and keyboard, but this sharing occurs only within your device.

Face ID / Touch ID

Biometric data is processed within Apple's Secure Enclave and is never accessed by this app.

We only receive the authentication result (success/failure).

Data Storage

All data is stored in local storage on your device.

No iCloud sync or cloud backup is performed.
